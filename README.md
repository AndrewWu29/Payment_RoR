## oPay歐付寶 全方位(All In One)金流介接 Ruby版

### 1. 介紹

* oPay歐付寶對於有收款需求的會員，提供完整的交易介接API，並有多種收款方式(All In One)可選擇使用。
本套件為Ruby版，可使用作建立訂單，接受付款通知，查詢訂單等金流交易的應用。
* 收款方式清單：
  * 信用卡(一次付清、分期付款、定期定額)
  * oPay儲值消費   
  * 網路ATM
  * ATM櫃員機
  * 超商代碼
  * 銀行快付
  * 微信支付

### 2. 安裝環境

* Ruby 2.3.1 (x64) 以上
